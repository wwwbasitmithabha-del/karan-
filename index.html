⚡ Google Sheets Backend
Connect once and your data syncs across all devices automatically.
🔧 Setup Guide
Follow these steps to connect your Google Sheet as a backend database.
STEP 1
Create a new Google Sheet at sheets.google.com. Name it CA StudyDesk DB.
STEP 2
Open Extensions → Apps Script. Delete all default code and paste the Code.gs script provided. Replace YOUR_SHEET_ID with your Sheet's ID from the URL.
STEP 3
In Apps Script, run the setupSheets() function once to create all 5 sheet tabs automatically.
STEP 4
Click Deploy → New Deployment → Web App. Set Execute as: Me, Access: Anyone. Copy the Web App URL.
STEP 5
Paste that URL in the login screen above and click Enter StudyDesk. Done! 🎉
